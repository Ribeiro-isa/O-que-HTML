O que é HTML?

HTML é uma linguagem de marcação usada para estruturar e apresentar conteúdo na web. Diferente de linguagens de programação, o HTML não é utilizado para criar funcionalidades lógicas, mas sim para organizar o conteúdo que será exibido no navegador.

Estrutura Básica de um Documento HTML

Todo documento HTML segue uma estrutura básica, que inclui elementos essenciais:


<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título da Página</title>
</head>
<body>
    <!-- Conteúdo da página vai aqui -->
</body>
</html>
  • <!DOCTYPE html>: Declaração do tipo de documento. Informa ao navegador que o documento está usando HTML5.
  • <html>: Elemento raiz que engloba todo o conteúdo da página.
  • <head>: Contém metadados, como o título da página e links para CSS ou scripts.
  • <meta charset="UTF-8">: Define a codificação de caracteres do documento.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Configura a visualização em dispositivos móveis.
  • <title>: Define o título que aparecerá na aba do navegador.
  • <body>: Onde o conteúdo visível da página é colocado.

Elementos HTML

Elementos HTML são os blocos de construção de uma página da web. Eles são representados por tags. Alguns exemplos comuns incluem:

  • <h1> a <h6>: Títulos, com <h1> sendo o mais importante e <h6> o menos.
  • <p>: Parágrafo de texto.
  • <a>: Link de hipertexto.
  • <img>: Imagem.
  • <ul>, <ol>, <li>: Listas não ordenadas, ordenadas e itens de lista, respectivamente.
  • <div>: Container genérico, usado para agrupar outros elementos.
  • <span>: Container em linha, utilizado para estilização de parte de texto.

Atributos HTML

Atributos fornecem informações adicionais sobre elementos. São definidos dentro da tag de abertura do elemento.

Exemplo:


<a href="https://example.com" target="_blank">Visite o Example</a>
  • href: Define o URL para onde o link aponta.
  • target: Especifica onde abrir o documento vinculado (por exemplo, _blank para uma nova aba).

Semântica HTML

A semântica refere-se ao uso apropriado de elementos HTML para representar o significado do conteúdo. O uso correto melhora a acessibilidade e SEO. Exemplos de elementos semânticos incluem:

  • <header>: Cabeçalho da página ou seção.
  • <nav>: Navegação.
  • <article>: Conteúdo independente, como um artigo de blog.
  • <section>: Agrupamento de conteúdo relacionado.
  • <footer>: Rodapé da página ou seção.

Formulários em HTML

Formulários permitem a coleta de dados do usuário. São compostos de vários elementos de entrada:

  • <form>: Elemento que agrupa todo o formulário.
  • <input>: Campo de entrada de dados, que pode ser do tipo text, password, email, etc.
  • <label>: Rótulo associado a um campo de entrada.
  • <textarea>: Campo de texto longo.
  • <button>: Botão de envio ou ação.

Exemplo de um formulário simples:


<form action="/enviar-dados" method="post">
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome">
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email">
    
    <button type="submit">Enviar</button>
</form>

Multimídia em HTML

HTML permite a inserção de multimídia, como imagens, vídeos e áudio.

  • <img>: Insere uma imagem. Atributos comuns incluem src (fonte) e alt (texto alternativo).
  • <video>: Insere um vídeo. Pode ter atributos como controls, autoplay, loop.
  • <audio>: Insere áudio, também com atributos como controls.

Exemplo de uso:


<img src="imagem.jpg" alt="Descrição da imagem">
<video src="video.mp4" controls></video>
<audio src="audio.mp3" controls></audio>

Links e Navegação

Os links são criados usando a tag <a>. Eles podem levar a outras páginas, seções na mesma página, arquivos para download, entre outros.

Exemplo:


<a href="https://example.com">Visite o Example</a>
<a href="#secao1">Vá para a Seção 1</a>

Tabelas em HTML

As tabelas são usadas para apresentar dados tabulares. Principais elementos:

  • <table>: Define uma tabela.
  • <tr>: Define uma linha de tabela.
  • <td>: Define uma célula de dados.
  • <th>: Define uma célula de cabeçalho.

Exemplo de tabela:


<table>
    <tr>
        <th>Nome</th>
        <th>Idade</th>
    </tr>
    <tr>
        <td>João</td>
        <td>30</td>
    </tr>
</table>

Boas Práticas em HTML

  • Validação: Use validadores HTML para garantir que seu código segue os padrões web.
  • Acessibilidade: Adicione atributos como alt em imagens e use elementos semânticos para melhorar a acessibilidade.
  • SEO: Utilize elementos semânticos e meta tags corretamente para otimizar seu site para motores de busca.
  • Mobile-first: Adote práticas de design responsivo, utilizando o <meta viewport> e media queries no CSS.

Evolução e Versões do HTML

O HTML passou por várias versões, começando pelo HTML 1.0 até o HTML5, que é a versão atual e mais amplamente usada. HTML5 trouxe novos elementos semânticos, suporte nativo a multimídia, APIs de JavaScript, entre outras funcionalidades avançadas.
